# Tugas ABP Telkom - Overview

Repository ini berisi kumpulan tugas mata kuliah **Aplikasi Berbasis Platform (ABP)** di Telkom University.
Kontennya mencakup latihan bertahap mulai dari HTML, CSS, JavaScript, PHP, AJAX, sampai pengenalan web framework (Laravel).

## Daftar Project

1. **Tugas 1 HTML**
	- Latihan dasar pembuatan halaman web menggunakan HTML.

2. **Tugas 2 CSS**
	- Styling halaman web menggunakan CSS (`index.html` dan `style.css`).

3. **Tugas 3 JS**
	- Implementasi dasar JavaScript pada halaman web (`Tugas 1.html`, `Tugas 2.html`).

4. **Tugas 3.1 - PHP**
	- Dasar pemrograman web dengan PHP, termasuk halaman `login` dan `register`.

5. **Tugas 3.2 - AJAX**
	- Implementasi komunikasi asynchronous menggunakan AJAX (`ajax.js`, `reply.php`).

6. **Tugas 4 - Web Framework 1**
	- Pengenalan framework Laravel (struktur project, route, controller, view, model).

7. **Tugas 5 - Web Framework 2**
	- Lanjutan latihan Laravel untuk memperdalam konsep framework pada pengembangan web.

## Tujuan Repository

- Mendokumentasikan progres pembelajaran ABP dari tugas awal hingga framework.
- Menjadi referensi implementasi tiap topik praktikum.
- Menyimpan hasil tugas secara terstruktur dalam satu workspace.

## Catatan

Setiap folder merepresentasikan satu topik/penugasan praktikum dan dapat dijalankan/ditinjau secara terpisah sesuai kebutuhan materi.
